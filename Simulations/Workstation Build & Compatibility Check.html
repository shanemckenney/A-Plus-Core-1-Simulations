A+ Core 1 – Workstation Build (Gaming + Family) + Compatibility Check

Build two Windows desktops based on the customer requirements. Select appropriate components and peripherals for each workstation.

Scenario & Requirements

Customer request

A customer wants two new desktops. The first is a gaming workstation. The second is a family workstation.

Gaming workstation requirements

• Play the newest games at a high frame rate
• Fast game load times
• Enough storage for several games installed at once
• High-end audio
• No concern about cost
• Run the current Windows OS

Family workstation requirements

• Word processing, videoconferencing, basic web surfing
• Minimal cost (as long as it meets the requirements)
• Run the current Windows OS

What you are being tested on

Choose parts that match the use case (performance vs cost), and identify compatibility issues that would prevent running the current Windows OS.
(Example: selecting an ARM CPU in a typical consumer Windows desktop scenario.)

Build Selection

Gaming Workstation

Goal: max performance
Tip: gaming has “high-end audio” as an explicit requirement.

Family Workstation

Goal: meet needs cheaply
Tip: videoconferencing requires a microphone (and typically a webcam).
